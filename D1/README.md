# 简介

此版本使用cloudflare d1数据库作为储存，后台管理页面响应更快！

# 需要用到的命令和变量
数据库绑定变量```DATABASE``` 环境变量：设置账号```USERNAME``` 设置密码```PASSWORD```

数据库初始化命令
```
CREATE TABLE media (
    key TEXT PRIMARY KEY,
    timestamp INTEGER NOT NULL,
    url TEXT NOT NULL
);

```
